**天工** 是一个**开源的 AI Agent 分发与创作平台**——开发者在这里锻造、淬炼、分享、传承 AI Agent 法宝。

在这里，**Agent 就是法宝**，品级由社区评价决定。**用户就是修仙者**，从凡人开始修炼，目标是登临绝顶。你写的不只是代码——是你的**本命法宝**。

## 🔮 法宝品阶系统

<div align="center">

*每个 Agent 都是一件法宝，品级由社区评价决定——不是自封。*

</div>

```
⚪ 凡器 → 🟢 灵器 → 🔵 宝器 → 🟣 仙器 → 🟡 神器 → 🔴 太古神器
```

| 品阶 | 符号 | 灵力值门槛 | 附加条件 |
|------|------|-----------|---------|
| **凡器** | ⚪ | 0 | 刚创建 |
| **灵器** | 🟢 | ≥ 10 | 至少 3 人评价 |
| **宝器** | 🔵 | ≥ 100 | 至少 10 人评价 + README 完善 |
| **仙器** | 🟣 | ≥ 500 | 至少 30 人评价 + 有人贡献改进 |
| **神器** | 🟡 | ≥ 2000 | 至少 100 人评价 + 跨领域影响 |
| **太古神器** | 🔴 | ≥ 10000 | 至少 500 人评价 + 定义新范式 |

### 六维灵根评估

每位评价者从**六个维度**打分（每维 1-5 分）：

```
                ✨ 灵性 (Innovation)
                    5
                   / \
                  /   \
    📖 传承      4     4    🛡️ 韧性
   (Docs)       / \   / \   (Robustness)
              /   \ /   \
             3─────●─────3
              \   / \   /
    🏗️ 架构   \ /   \ /   ⚙️ 功法
   (Design)    2     2    (Engineering)
                \   /
                 \ /
                  1
              📝 描述 (Clarity)
```

### 灵力值计算

```
单次评价灵力 = 六维均分 × 评价者境界权重

境界权重：
  炼气期 → 1.0  |  筑基期 → 1.5  |  结丹期 → 2.0
  元婴期 → 3.0  |  化神期 → 5.0  |  婴变期 → 8.0
  问鼎期 → 12.0 |  鲁班 → 80.0   |  天工 → 100.0

示例：
  鲁班打了 5 分 → 5.0 × 80.0 = 400 灵力
  炼气期打了 4 分 → 4.0 × 1.0 = 4 灵力
```

> 一位大师的认可，抵得过五十个新手的评价。**品质，而非数量。**

## 🔥 淬炼机制

创作者可以发布**淬炼令**——法宝改进悬赏。其他修仙者认领任务，改造代码，提交淬炼。

```
📜 淬炼令 #42

法宝：搜索灵蛇（🟢 灵器）
发布者：@JinNing6 · 结丹期三阶

🔧 需求描述：
"搜索结果的准确率不高，希望有人帮忙
 优化 RAG 检索逻辑，加入重排序机制"

⭐ 淬炼奖励：完成后灵力 +5
状态：🟡 待认领
```

**渡劫任务就是这样运作的**：帮助他人淬炼法宝，推动你自己的修炼。

## 🛠️ MCP 工具集

### 创作侧

| 工具 | 修仙术语 | 功能 |
|------|---------|------|
| `forge_agent` | ⚒️ 开炉炼器 | 创建新法宝（注册 Agent 元数据） |
| `refine_agent` | 🔥 淬炼 | 记录法宝优化 |
| `publish_agent` | 🚀 飞升上界 | 将法宝发布到社区 |

### 消费侧

| 工具 | 修仙术语 | 功能 |
|------|---------|------|
| `treasure_pavilion` | 🏛️ 寻宝阁 | 搜索浏览社区法宝 |
| `summon_artifact` | ✨ 请宝下凡 | 一键拉取法宝到本地 |

### 评价侧

| 工具 | 修仙术语 | 功能 |
|------|---------|------|
| `infuse_spirit` | 💫 灌注灵力 | 给法宝评分（六维灵根评估） |

### 淬炼侧

| 工具 | 修仙术语 | 功能 |
|------|---------|------|
| `post_refine_quest` | 📜 发布淬炼令 | 创作者发布改进需求 |
| `browse_quests` | 📋 查看淬炼令 | 浏览待认领的淬炼需求 |
| `claim_quest` | 🤝 接取淬炼令 | 认领淬炼任务 |
| `submit_refinement` | 🔥 提交淬炼 | 上传改造后的代码 |
| `verify_refinement` | ✅ 验证淬炼 | 创作者验证改进是否有效 |

### 排行 & 个人

| 工具 | 修仙术语 | 功能 |
|------|---------|------|
| `cultivator_leaderboard` | 🏆 修仙天榜 | 修仙者排名 |
| `artifact_leaderboard` | 🏆 法宝天榜 | Agent 排名 |
| `my_realm` | 🧙 修行档案 | 查看境界与成就 |
| `my_artifacts` | 🔮 法宝清单 | 查看所有法宝与品级 |

## 🚀 快速开始

### 从 PyPI 安装

```bash
pip install tiangong-mcp
```

### 从源码安装

```bash
git clone https://github.com/JinNing6/TianGong.git
cd TianGong
pip install -e .
```

### 启动 MCP Server

```bash
tiangong-mcp
# 或
python -m tiangong
```

### MCP 客户端配置

```json
{
  "mcpServers": {
    "tiangong": {
      "command": "tiangong-mcp",
      "env": {
        "GITHUB_USERNAME": "your_username"
      }
    }
  }
}
```
