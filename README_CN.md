## 🎯 天工是什么？

**天工** 是一个**开源的 AI Agent 修炼平台**——开发者在这里创建、淬炼、突破、分享自己的 Agent。

> **天工**源自明代宋应星《天工开物》——以天赐造物之力，开启万物本质。
>
> 但天工的精神内核不止于"造"——更在于**"逆"**。
>
> 没有大厂的算力？没有顶级团队？没有巨额资金？
> **那又如何？** 以凡人之躯，铸逆天之器——这就是天工修仙者的信条。

你不是在写代码——你是在**锻造本命法宝**。

## 🚀 快速开始

### 从 PyPI 安装

```bash
pip install tiangong-mcp
```

### 从源码安装

```bash
git clone https://github.com/JinNing6/TianGong.git
cd TianGong
pip install -e .
```

### 启动 MCP Server

```bash
tiangong-mcp
# 或
python -m tiangong
```

### MCP 客户端配置

```json
{
  "mcpServers": {
    "tiangong": {
      "command": "tiangong-mcp",
      "env": {
        "GITHUB_USERNAME": "your_username"
      }
    }
  }
}
```
