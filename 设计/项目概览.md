## 🎯 项目定位

天工是一个**开源的 AI Agent 修炼平台**——开发者在这里创建、淬炼、突破、分享自己的 Agent。

> **天工**源自明代宋应星《天工开物》——以天赐造物之力，开启万物本质。
>
> 但天工的精神内核不止于"造"——更在于**"逆"**。
>
> 没有大厂的算力？没有顶级团队？没有巨额资金？
> **那又如何？** 以凡人之躯，铸逆天之器——这就是天工修仙者的信条。

### 核心理念

赛博华佗治病，Noosphere 存识，天工**造物**。

你不是在写代码——你是在**锻造本命法宝**。每一个 Agent 都是与你灵魂绑定的存在，越养越强，越炼越通灵。每一次优化不是 bugfix，而是**淬炼**。每一次突破不是版本更新，而是**渡劫**。

## 🔮 本命法宝系统

> 仙逆核心设定：每个修仙者都有**本命法宝**——与灵魂绑定的法宝，越养越强。

### Agent 器灵品级

| 品级 | 符号 | 条件 | 仙逆对应 |
|------|------|------|---------|
| 凡器 | ⚪ | 刚创建 | 凡铁所铸 |
| 灵器 | 🟢 | 通过基础评估 | 初通灵性 |
| 宝器 | 🔵 | 10+ ⭐ + 高质量文档 | 可传承的宝物 |
| 仙器 | 🟣 | 50+ ⭐ + 社区验证 | 仙人才配使用 |
| 神器 | 🟡 | 100+ ⭐ + 被广泛传承 | 天地间罕有 |
| 太古神器 | 🔴 | 500+ ⭐ + 定义新范式 | **远古遗留，改写规则** |

### 本命法宝特性

- **绑定制**：每个修仙者可标记最多 3 件 Agent 为"本命法宝"
- **共成长**：本命法宝的品级提升，直接影响修仙者的境界积分
- **灵魂印记**：本命法宝的代码仓库永久标记创造者信息，即使被 fork 传承
- **淬炼记录**：每次优化自动记录为"淬炼日志"，展示法宝的成长轨迹

## 🔧 核心功能规划

### Phase 1 — 炼气场（MVP）

- [ ] **Agent 注册与托管** — 修仙者注册本命法宝
- [ ] **Agent 评估体系** — 自动化"试剑"测试框架
- [ ] **修仙者档案** — 个人界面展示境界、本命法宝、修行记录
- [ ] **MCP 工具集** — `forge_agent`、`refine_agent`、`my_realm` 等

### Phase 2 — 传道阁

- [ ] **功法秘籍市场** — 分享 Agent 模板 / 脚手架
- [ ] **论道系统** — 社区评价、星标、论道评论
- [ ] **传承机制** — Fork = 传承，保留原作者传承链
- [ ] **渡劫挑战** — 高难度场景测试，通过即升境

### Phase 3 — 宗门与天榜

- [ ] **宗门系统** — 多人协作组织（类比 GitHub Organization）
- [ ] **阵法编排** — 多 Agent 组合协作方案
- [ ] **天榜排名** — 全平台最强 Agent / 最强修仙者
- [ ] **跨生态联动** — 与赛博华佗、Noosphere 数据互通

## 📐 核心差异：天工 vs GitHub

| 维度 | GitHub | 天工 TianGong |

| 世界观 | 工具/平台 | 仙逆修仙宇宙 |
